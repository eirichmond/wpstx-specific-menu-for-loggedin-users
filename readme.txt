== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wpstx-specific-menu-for-loggedin-users.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php do_action('wpstx_users_menu'); ?>` in your templates
